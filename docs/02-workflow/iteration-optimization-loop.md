# 迭代优化闭环（PM / 产评）

## 目标

把“本周执行结果”转化成“下周更稳的系统能力”。

## 周复盘输入

- Issue 完成率
- 平均 lead time
- Blocked 总时长
- 返工次数
- 发布后缺陷数

## 复盘输出

每周至少输出三项：

1. 删除一条低价值流程
2. 增加一条高价值约束
3. 优化一个模板字段

## 指标建议

### 交付效率

- Issue 完成率（目标 >= 80%）
- P0 准时率（目标 = 100%）
- 平均从 Ready 到 Done 天数

### 质量稳定性

- 验收不通过率
- 发布后回滚次数
- 关键链路失败次数

### 协作健康度

- 跨端阻塞数量
- 无验收标准 Issue 占比（目标 = 0）
- 无 Spec 链接 Issue 占比（目标 = 0）

## 月度升级动作

- 月初：清理过期规范
- 月中：评估 Harness 覆盖度
- 月末：升级 AGENTS 与模板版本
